Инструкция по подключению на iOS

Соединения типа IPSec/IKEv2 MSCHAPv2 EAP является полностью безопасным, с проверкой подлинности сервера VPN. Именно для идентификации сервера требуется сначала установить корневой сертификат VPN сервера. После этого ваша операционная система начнёт доверять VPN серверу.

Сначала нужно импортировать корневой сертификат выполнив следующие шаги:

  1. Загрзите корневой сертификат к себе на устройство ( скачать).
  2. Нажмите на скаченный файл корневого сертификата, затем нажмите Установить и введите код доступа. После установки нажмите Готово.

Импорт сертификата завершён, осталось выполнить настройку подключения VPN, для чего выполните следующие шаги:

  1. Откройте Настройки -> Общие -> VPN и нажмите Добавить конфигурацию VPN. После этого откроется экран конфигурации подключения VPN.
  2. Нажмите Тип и выберите IKEv2.
  3. В поле Описание введите короткое имя подключения VPN. Вы можете выбрать все, что угодно.
  4. В поле Server (сервер) и Remote ID (удаленный идентификатор) введите - "vpn.thebestsoft.ru". Поле Local ID (локальный идентификатор) нельзя оставлять пустым.
  5. Введите имя пользователя и пароль в разделе Аутентификация, а затем нажмите Готово.
  6. Выберите соединение VPN, которое вы только что создали, нажмите переключатель вверху страницы, и подключение будет установлено.

Поздравляю! Настройка завершена.
