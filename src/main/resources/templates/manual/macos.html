Инструкция по подключению на iOS

Соединения типа IPSec/IKEv2 MSCHAPv2 EAP является полностью безопасным, с проверкой подлинности сервера VPN. Именно для идентификации сервера требуется сначала установить корневой сертификат VPN сервера. После этого ваша операционная система начнёт доверять VPN серверу.

Сначала нужно импортировать корневой сертификат выполнив следующие шаги:

  1. Загрзите корневой сертификат к себе на устройство ( скачать).
  2. Дважды щелкните файл сертификата. Откроется экран «Keychain Access» с диалоговым окном, где будет указано: «Keychain Access is trying to modify the system keychain. Enter your password to allow this».
  3. Введите свой пароль и нажмите Modify Keychain.
  4. Дважды щелкните на импортированный сертификат VPN. После этого откроется небольшое окно свойств, где вы сможете указать уровни доверия. Укажите для параметра IP Security (IPSec) значение Always Trust (всегда доверять), после чего вам нужно будет снова ввести пароль. Настройка будет сохранена автоматически после ввода пароля.

Импорт сертификата завершён, осталось выполнить настройку подключения VPN, для чего выполните следующие шаги:

  1. Откройте раздел System Preferences (настройки системы) и выберите пункт Network (сеть).
  2. Нажмите небольшую кнопку «плюс» в нижнем левом углу списка сетей.
  3. В открывшемся всплывающем окне установите для параметра Interface (интерфейс) значение VPN, установите для параметра VPN Type (тип VPN) значение IKEv2 и присвойте имя соединению.
  4. В поле Server (сервер) и Remote ID (удаленный идентификатор) введите доменное имя или IP-адрес сервера. Оставьте поле Local ID (локальный идентификатор) пустым.
  5. Нажмите Authentication Settings (настройки аутентификации), выберите пункт Username (имя пользователя) и введите имя пользователя и пароль, которые вы настроили для своего пользователя VPN. Затем нажмите OK.
  6. Нажмите Authentication Settings (настройки аутентификации), выберите пункт Username (имя пользователя) и введите имя пользователя и пароль. Затем нажмите OK.
  7. Наконец, нажмите кнопку Connect (подключение) для подключения к VPN. Теперь вы должны быть подключены к VPN.

Поздравляю! Настройка завершена.
